Privacy Policy for Sofia Text Summarizer

Effective Date: 30 January 2026

1. Data Collection

This extension does not collect, store, or share any personal data.

2. Text Processing

When you use the summarization feature:

  • Your text is sent securely to our proxy server
  • The proxy server forwards it to DeepSeek API for processing
  • Text is not stored on any server
  • Text is not shared with third parties (except DeepSeek API)

3. API Usage

We use DeepSeek API for text summarization. Their privacy policy applies to data sent to their service.

4. Local Storage

The extension may store settings locally in your browser using Chrome Storage API.

5. Contact

For questions: your-email@example.com
